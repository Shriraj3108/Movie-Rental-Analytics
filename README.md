# Shakila_DVD_Store_Rental_Analytics:
## Overview:
The Movie Rental Analytics Capstone Project analyzes the Sakila DVD Store dataset using Power BI to uncover insights into rental performance, revenue trends, customer behavior, film demand, and actor contribution. The dashboards highlight geographic revenue distribution, seasonal rental patterns, high-performing genres, and the impact of actor participation on overall revenue. By examining rental duration, customer engagement, and category-level performance, the project identifies key drivers influencing business growth and provides data-driven insights to support strategic decision-making and revenue optimization.
